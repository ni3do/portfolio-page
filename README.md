# My Portfolio

My deployed portfolio page, wrapped as a docker container to run on Unraid

# Run for yourself

Clone the repo:

    git clone git@github.com:ni3do/portfolio-page.git

## Run without docker

Install dependencies and start the project:

    cd app
    yarn install
    yarn dev

## Run with docker

    docker-compose build
    docker-compose up
